🛒 Blinkit Sales Dashboard (Data Analysis Project)

📌 Project Overview

This project presents an interactive sales analysis dashboard for Blinkit – India's Last Minute App.
The dashboard visualizes key business metrics such as total sales, average sales, number of items, and average rating, helping stakeholders quickly understand sales performance across outlets, item categories, and outlet types.
The goal of this project is to demonstrate data analysis and dashboarding skills by transforming raw sales data into meaningful business insights.

📊 Dashboard Preview

🎯 Key Metrics Displayed
The dashboard highlights the following KPIs:

  1. Total Sales: $1.20M
	
  2. Average Sales: $141
	
  3. Number of Items: 8523
	
  4. Average Rating: 3.9

These KPIs help quickly evaluate the overall performance of Blinkit's product sales.

📈 Dashboard Insights

1️⃣ Sales by Item Type

Fruits & Snacks generate the highest revenue (~$0.18M).

Household and Frozen products also contribute significantly.

Breakfast and Seafood show the lowest sales contribution.

2️⃣ Sales by Fat Content

Regular fat items dominate sales ($776K).

Low-fat items contribute $425K.

3️⃣ Sales by Outlet Tier

Tier 3 outlets generate the highest sales (~$472K).

Tier 2 and Tier 1 follow with moderate contributions.

4️⃣ Sales by Outlet Size

Medium outlets produce the highest sales ($507.9K).

Small outlets contribute less compared to medium outlets.

5️⃣ Sales by Outlet Type

Supermarket Type1 dominates the total sales.
